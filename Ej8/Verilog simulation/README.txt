Bienvenido a la simulacion!

-Para compilar: "make"
-Para correr el programa: "make run"
-Para ver la simulacion en GTKWave: "make view"
-Si se realizan cambios en el programa, para cargar cambios con GTKWave abierto: "make reload"

Visualizacion en GTKWave:
  Se recomienda cargar visualizar las variables en el siguiente orden para una correcta comprension:
	1. clk1
	2. clk2
	3. comp
	4. countInt
	5. validCountInt
	6. clk3
	7. showedCountInt
	8. sevenSegOutInt
